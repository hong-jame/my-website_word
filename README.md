# 중2 영어 단어장 · 말하기 시험 앱

자이스토리 중등 영어 독해 기본 2 (pp.74~161) · 33세트 · 920단어

## 기능
- **단어 학습**: 단어·뜻·예문 보기 + 발음 듣기(영어), 즐겨찾기(★)
- **뜻 말하기 시험 (영어→한글)**: 영어 단어를 보고 한글 뜻을 말하거나 입력하면 채점
- **단어 말하기 시험 (한글→영어)**: 한글 뜻을 보고 영어 단어를 말하거나 입력하면 채점
- **틀린 단어만 다시 풀기**, **최고 점수 기록**, **학습 기록 화면**
- **부모님께 결과 보내기**: 시험 결과를 카톡·문자로 공유

## 깃허브 페이지로 배포하는 법
1. 새 저장소(Repository)를 만든다 (Public).
2. 이 폴더의 `index.html` 파일을 업로드한다.
3. 저장소 **Settings → Pages** 로 이동.
4. Source 를 **main 브랜치 / (root)** 로 지정하고 저장.
5. 1~2분 뒤 생기는 `https://(내아이디).github.io/(저장소이름)/` 링크로 접속.

## 사용 팁
- 마이크(음성 인식)는 **크롬(안드로이드·PC)·사파리(아이폰)**에서 동작하며,
  깃허브 페이지(https)로 열면 정상 작동합니다.
- 첫 사용 시 **마이크 권한 "허용"**을 눌러야 합니다.
- 학습 기록·즐겨찾기는 각자 기기 브라우저에 저장됩니다.
